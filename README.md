# edit-script

Edit npm scripts from the command line without worrying about json escaping.

[![asciicast](https://asciinema.org/a/92481.png)](https://asciinema.org/a/92481)

## Installation

```bash
npm install edit-script
```

## Usage

```
$ edit-script --help
Edit npm scripts from the command line without worrying about json escaping.

    edit-script
    edit-script <script>
```
